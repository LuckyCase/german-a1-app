# German A1 Telegram Bot 🇩🇪

Telegram-бот для изучения немецкого языка уровня A1 по стандартам Goethe-Institut.

**Чат бота** используется только как вход: команда `/start` и кнопка «Открыть приложение». Вся учёба, настройки и прогресс — в **Telegram Web App** (страница `web_server.py`).

## ✨ Функции

- **📚 Карточки (Flashcards)** - изучение 262 слов с выбором перевода и примерами
- **📝 Грамматические тесты** - 8 тестов по темам A1 с теорией и объяснениями
- **💬 Фразы** - изучение полезных фраз для повседневных ситуаций
- **🗣️ Диалоги** - интерактивные диалоги с упражнениями для практики разговорной речи
- **🔊 Аудио произношение** - озвучка немецких слов и фраз через gTTS
- **📊 Трекер прогресса** - детальная статистика изучения по всем разделам
- **⏰ Ежедневные напоминания** - сообщения в чат с кнопкой открытия приложения (job в `python -m bot.main`; настройка времени — через БД/API, в Web App UI пока может не быть)
- **🌐 Web App** - современный веб-интерфейс прямо в Telegram с адаптивным дизайном

## Установка локально

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Windows
   source venv/bin/activate  # Linux/Mac
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Создайте файл `.env`:
   ```env
   TELEGRAM_BOT_TOKEN=your_token_from_botfather
   DATABASE_URL=postgresql://user:pass@host:5432/dbname  # Для продакшн
   # Или оставьте пустым для SQLite (локальная разработка)
   WEB_APP_URL=http://localhost:5000  # Для локального теста
   ```
5. Инициализируйте базу данных (опционально, создастся автоматически):
   ```bash
   python -c "import asyncio; from bot.database import init_db; asyncio.run(init_db())"
   ```
6. Запустите бота:
   ```bash
   python -m bot.main
   ```
   
   Или для Web App режима:
   ```bash
   # Терминал 1: Веб-сервер
   python web_server.py
   
   # Терминал 2: Бот (опционально, если нужен polling)
   python -m bot.main
   ```

## 🚀 Деплой на Render

### Вариант 1: Web Service (рекомендуется)

Для работы Web App и Webhook:

1. Создайте репозиторий на GitHub и загрузите код
2. Зайдите на [render.com](https://render.com)
3. Создайте новый **Web Service** (не Background Worker!)
4. Подключите GitHub репозиторий
5. Настройте:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python web_server.py`
6. Добавьте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` = ваш токен от BotFather
   - `DATABASE_URL` = PostgreSQL URL (можно использовать Supabase/Neon бесплатно)
   - `WEB_APP_URL` = URL вашего Render сервиса (автоматически)
7. После деплоя настройте Webhook:
   - Откройте `https://ваш-сервис.onrender.com/setup-webhook`
   - Проверьте статус: `https://ваш-сервис.onrender.com/webhook-info`
   - Проверьте здоровье сервиса: `https://ваш-сервис.onrender.com/health`

### Вариант 2: Background Worker (только бот)

Если нужен только бот без Web App:

1. Создайте **Background Worker**
2. **Start Command**: `python -m bot.main`
3. Добавьте `TELEGRAM_BOT_TOKEN` и `DATABASE_URL`

**Примечание:** Для Web App нужен Web Service, так как требуется HTTP endpoint. Webhook настраивается автоматически через `/setup-webhook` endpoint.

### Проверка работы после деплоя

1. **Проверка Webhook:**
   ```
   https://ваш-сервис.onrender.com/webhook-info
   ```

2. **Проверка здоровья сервиса:**
   ```
   https://ваш-сервис.onrender.com/health
   ```

3. **Отладка (если есть проблемы):**
   ```
   https://ваш-сервис.onrender.com/debug
   ```

## 💬 Использование

Основной функционал доступен через **Web App** - современный веб-интерфейс прямо в Telegram.

**Как начать:**
1. Откройте бота в Telegram
2. Отправьте команду `/start`
3. Нажмите кнопку "🚀 Открыть приложение"
4. Начните изучать немецкий язык!

## 🛠️ Разработка

### Добавление нового контента

1. **Добавить слова в категорию:**
   - Откройте `data/A1/vocabulary/название_категории.json`
   - Добавьте новый объект в массив `words`:
   ```json
   {
     "de": "das Wort",
     "ru": "слово",
     "example": "Das ist ein Wort.",
     "example_ru": "Это слово."
   }
   ```
   - Перезапустите бота (кэш обновится автоматически)

2. **Добавить грамматический тест:**
   - Создайте `data/A1/grammar/название_темы.json`
   - Используйте структуру из существующих файлов
   - Добавьте теорию в поле `theory` (опционально)

3. **Добавить фразы:**
   - Откройте `data/A1/phrases/название_категории.json`
   - Добавьте новый объект в массив `phrases`:
   ```json
   {
     "de": "Ich hätte gern...",
     "ru": "Я бы хотел...",
     "context": "Контекст использования",
     "example": "Пример диалога",
     "example_ru": "Перевод примера"
   }
   ```

4. **Добавить диалог:**
   - Создайте `data/A1/dialogues/название_темы.json`
   - Используйте структуру из существующих файлов
   - Добавьте упражнения в поле `exercises` (опционально)

5. **Перезагрузка контента:**
   ```python
   from bot.content_manager import reload_content
   reload_content()  # Очистит кэш и перезагрузит данные
   ```

## 📊 База данных

Проект использует **PostgreSQL** (asyncpg) для продакшн и поддерживает SQLite для локальной разработки.

**Таблицы:**
- `users` - пользователи бота
- `progress` - прогресс изучения слов
- `grammar_results` - результаты грамматических тестов
- `phrase_progress` - прогресс изучения фраз
- `dialogue_progress` - прогресс изучения диалогов
- `daily_stats` - ежедневная статистика

## 🔧 Технологии

- **Python 3.10+**
- **python-telegram-bot** - Telegram Bot API (Webhook режим)
- **asyncpg** - PostgreSQL драйвер (async)
- **Flask** - веб-сервер для Web App и Webhook
- **gTTS** - генерация аудио произношения
- **JSON** - хранение учебных данных
- **Gunicorn** - WSGI сервер для продакшн
- **nest-asyncio** - поддержка вложенных event loops

## 📚 Контент

### Словарь (Vocabulary)
- **262 слова** в 11 категориях:
  - Личная информация (14 слов)
  - Семья (15 слов)
  - Еда и напитки (24 слова)
  - Дом и жильё (18 слов)
  - Время (22 слова)
  - Важные глаголы (24 слова)
  - Транспорт и город (16 слов)
  - Покупки (14 слов)
  - Числа (20 слов)
  - Прилагательные (20 слов)
  - Важные фразы (15 слов)

- Каждое слово включает:
  - Немецкое слово с артиклем
  - Русский перевод
  - Пример предложения (DE + RU)

### Грамматика (Grammar)
- **8 тестов** с **78 вопросами**:
  1. Артикли (der, die, das) - 10 вопросов
  2. Глаголы sein и haben - 10 вопросов
  3. Настоящее время (Präsens) - 10 вопросов
  4. Отрицание (nicht/kein) - 10 вопросов
  5. Порядок слов - 5 вопросов
  6. Аккузатив (Akkusativ) - 8 вопросов
  7. Модальные глаголы - 8 вопросов
  8. Вопросительные слова - 10 вопросов

- Каждый тест включает:
  - Теоретический материал
  - Примеры использования
  - Вопросы с вариантами ответов
  - Подробные объяснения

### Фразы (Phrases)
- **15 фраз** в 3 ситуативных категориях:
  - В кафе (5 фраз)
  - На вокзале (5 фраз)
  - У врача (5 фраз)

- Каждая фраза включает:
  - Немецкая фраза
  - Русский перевод
  - Контекст использования
  - Пример диалога (DE + RU)
  - Аудио произношение

### Диалоги (Dialogues)
- **2 диалога** с упражнениями:
  - Приветствия (Begrüßungen)
  - В ресторане (Im Restaurant)

- Каждый диалог включает:
  - Полный диалог с репликами персонажей
  - Перевод каждой реплики
  - Интерактивные упражнения для закрепления
  - Отслеживание прогресса выполнения
